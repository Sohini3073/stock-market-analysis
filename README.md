## 📌 Project Overview
A complete end-to-end stock market analysis of **5 major tech companies** using real historical data from Yahoo Finance. Covers the full data analytics pipeline — data collection, cleaning, EDA, technical indicators, ML forecasting, and NLP sentiment analysis.

---

## 🏢 Stocks Analyzed
| Company   | Ticker | Period |
|-----------|--------|--------|
| Apple     | AAPL   | 2021–2026 |
| Amazon    | AMZN   | 2021–2026 |
| Tesla     | TSLA   | 2021–2026 |
| Microsoft | MSFT   | 2021–2026 |
| Google    | GOOGL  | 2021–2026 |

---

## 🎯 What This Project Covers
- 📥 **Data Collection** — Real OHLCV data via `yfinance` API
- 🔍 **EDA** — Price trends, volume, daily returns, correlation heatmap
- 📐 **Technical Indicators** — SMA, RSI, Bollinger Bands, MACD
- 🔮 **Forecasting** — 90-day prediction using Facebook Prophet
- 💬 **Sentiment Analysis** — News headlines scored using VADER (NLP)
- 📊 **Interactive Dashboard** — Multi-panel dashboard with Plotly
